为什么选择 Markdown？

Markdown 是一种轻量级标记语言，它允许你使用易读易写的纯文本格式编写文档，然后转换为结构化的 HTML。 对于博客作者、程序员和技术写作者来说，Markdown 是 效率与优雅 的代名词。

核心优势

  • 纯文本 — 任何编辑器都能打开，无需担心格式兼容。
  • 易学习 — 语法极简，5 分钟即可上手。
  • 可移植 — 轻松转换为 PDF、HTML、Word 等格式。
  • 专注内容 — 让你聚焦于写作本身，而非排版。

“Markdown 是写给未来的情书。” — 佚名

它让写作回归本质，让内容在岁月中依然可读。

代码示例 · 简单优雅

下面是一段 JavaScript 函数，用于计算斐波那契数列：

function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

// 输出第 10 项
console.log(fibonacci(10)); // 55

你可以像这样使用 inline code 来标记变量或函数名。 也可以用 粗体 和 斜体 来强调重点。
